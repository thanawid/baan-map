ถ้ายังไม่ได้ใส่ค่า Firebase ใน firebase-config.js ระบบจะทำงานแบบ Local ก่อน

วิธีใช้งาน Baan-Map V2

ลุยงานสนามได้ แล้วค่อยซิงก์ขึ้น Firebase เหมือนข้าวเหนียวมีหมูปิ้งครบชุด
1
ตั้งค่า Firebase

ใส่ค่า firebaseConfig ในไฟล์ firebase-config.js แล้วเปิด Authentication + Firestore

2
อัปขึ้น GitHub Pages

อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ โดยให้ index.html อยู่หน้าแรกของ repo

3
เพิ่มบ้านจากแผนที่/GPS

กดเพิ่มบ้านแล้วแตะจุด หรือยืนหน้าบ้านแล้วกดปักหมุดจุดที่ยืน

4
ซิงก์และส่งออกรายงาน

เมื่อ Login แล้ว ระบบจะซิงก์ Firestore; ยังส่งออก JSON/CSV ได้เสมอ

แผนที่ตั้งศูนย์กลางที่บางรักน้อยแล้ว แต่แนวเขตหมู่ยังเป็นค่าโดยประมาณ ควรปรับด้วยโหมด “แก้เขตหมู่” หรือแทนด้วย GeoJSON ทางการภายหลัง
